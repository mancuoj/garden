## typing game

### 想法

- 打字游戏，锻炼英文阅读 + 单词能力
- 文章（双语），短文章（来源？）
- **在打字完成后出现单词中文翻译，整句打完出现整句翻译**
- CLI + Web 端
- 方便贡献文章，转 JSON 格式
- 需不需要后端？账号信息 + 存储 + 图表展示
- 多主题，多按键声音
- 手势导航，推荐手势
- 错误热力图 heatmap
- 手速展示
- 错误回顾
- 更多娱乐模式

  - 抓小偷
  - 钢琴键
  - qq 炫舞
