## Callouts

> [!tip] Title
>
> This is a callout!
> 
> > [!info] Nested
> > 
> > Yes, they can!

语法如下：

```
> [!tip] Title
>
> This is a callout!
> 
> > [!info] Nested
> > 
> > Yes, they can!
```

tip, info 可以替换，不同图标，不同颜色：

- info
- examples, example
- notes, note
- summaries, abstract, summary, tldr
- info, todo
- tip, hint, important
- success, check, done
- question, help, faq
- warning, caution, attention
- failure, fail, missing
- danger, error
- bug
- quote, cite
