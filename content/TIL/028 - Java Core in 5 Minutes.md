我的目的是速成然后直接上 Spring Boot 做一个简单的 CRUD 项目，所以搜了搜前置要求：

- Beginner Java Experience
- Java Build Tools (Maven or Gradle) 更推荐 Maven
- Java 17 JDK
- IDE 
  - 前期学语法可以 VS Code 过度
  - 后面还是老老实实 IDEA
  - 任意 API Testing Tool


## Install in WSL2

```sh
sudo apt update
sudo apt search jdk
sudo apt install openjdk-17-jdk
java --version
javac --version
```
